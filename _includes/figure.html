{%- comment -%} Figure with a LaTeX-style caption: small-caps auto-numbered lead-in ("Figure 1.") from a CSS counter, then the body. Usage: {% include figure.html src="/assets/images/foo/bar.png" width="800" caption="Some *markdown* caption with [links](https://example.com)." alt="Alt text for the image" %} src is required; caption, width, height, title, alt, and class are optional (class adds a modifier to the figure, e.g. "framed-grayscale"). Without a caption the figure renders unnumbered. Pass title="Figure 3b." to override the auto number. Multiple images (one row, shared caption): src="/a.png,/b.png,/c.png" The output is a single line so the figure also works inside list items. {%- endcomment -%} {%- assign figure_srcs = include.src | split: "," -%}
